]]> By: Sandeep Patil http://thegeekdiary.com/the-ultimate-veritas-cluster-server-vcs-interview-questions/#comment-8517 Sat, 20 Sep 2014 05:55:57 +0000 http://thegeekdiary.com/?p=2266#comment-8517 I guess you meant to make the passive node as active and vice versa in an active-passive vcs configuration. To do this, you need to switchover all your service groups from one node to other (requires downtime) and then to make these changes persists across cluster reboot, make sure you edit the main.cf and have the OY node in the AutoStartList attribute. AutoStartList attribute defines which nodes will automatically bring the service group online at VCS startup. Also, if required change the SystemList attribute accordingly. For other changes if any, I need to have a look at your main.cf.
