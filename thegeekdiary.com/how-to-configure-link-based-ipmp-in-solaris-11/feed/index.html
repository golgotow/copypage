]]> By: Sandeep Patil http://thegeekdiary.com/how-to-configure-link-based-ipmp-in-solaris-11/#comment-1573 Fri, 31 Jan 2014 19:44:42 +0000 http://thegeekdiary.com/?p=1231#comment-1573 why we are using only 1 IP address for the ipmp interfaces?
Link based IPMP uses interface kernel driver to check the status of the interface. whereas in probe based IPMP, ICMP probe messages on test address are sent to external targets to check the status of the interface. Thus in Link based IPMP we don’t assign any IP addresses to the interface as it does not depend on any external target to confirm that the interface has failed.

Read these articles to get more details
Solaris 10 Probe based IPMP
Solaris 10 Link Based IPMP
